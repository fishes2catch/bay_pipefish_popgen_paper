Reproducible analysis pipeline for:

**Currey MC, Woods MA, Goyal V, Bassham S, Healey HM, and Cresko WA. Genome-wide Analysis Reveals High Genomic Diversity and Panmixia in Bay Pipefish (*Syngnathus leptorhynchus*) from Coos Bay, Oregon. *Ichthyology and Herpetology*.**

---

## Overview

This repository contains all R and Quarto code used to generate the figures and statistical results reported in Currey et al. Bay pipefish (*Syngnathus leptorhynchus*) were collected by beach seine from two sites approximately 17 km apart in the Coos Bay estuary, Oregon, USA (Causeway and Valino Island) in September 2023. RAD-seq data from 116 individuals were used to assess population structure, genetic diversity, kinship, linkage disequilibrium, and effective population size.

## Quick Start

1. Clone this repository:
   \`\`\`bash
   git clone https://github.com/fishes2catch/bay_pipefish_popgen_paper.git
   cd bay_pipefish_popgen_paper
   \`\`\`
2. Install required R packages (see below)
3. Install required external software (see below)
4. Generate input files by following Section 0 of the Quarto document
5. Place input files in the \`data/\` directory
6. Render the document:
   \`\`\`bash
   quarto render analysis/bay_pipefish_popgen_analysis.qmd
   \`\`\`
