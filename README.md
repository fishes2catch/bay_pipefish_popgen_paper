# Bay Pipefish Population Genomics — Coos Bay, Oregon

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.XXXXXXX.svg)](https://doi.org/10.5281/zenodo.XXXXXXX)

Reproducible analysis pipeline for:

**Currey MC, Woods MA, Goyal V, Bassham S, Healey HM, and Cresko WA. Genome-wide Analysis Reveals High Genomic Diversity and Panmixia in Bay Pipefish (*Syngnathus leptorhynchus*) from Coos Bay, Oregon. *Ichthyology and Herpetology*.**

---

## Overview

This repository contains all R and Quarto code used to generate the figures and statistical results reported in Currey et al. Bay pipefish (*Syngnathus leptorhynchus*) were collected by beach seine from two sites approximately 17 km apart in the Coos Bay estuary, Oregon, USA (Causeway and Valino Island) in September 2023. RAD-seq data from 116 individuals were used to assess population structure, genetic diversity, kinship, linkage disequilibrium, and effective population size.

## Quick Start

1. Clone this repository
2. Install required R packages (see below)
3. Install required external software (see below)
4. Generate input files by following the HPC pipeline in `docs/pipeline_overview.md`
5. Place input files in `input/`
6. Render the analysis:

```bash
git clone https://github.com/fishes2catch/bay_pipefish_popgen_paper.git
cd bay_pipefish_popgen_paper
quarto render bay_pipefish_popgen_analysis.qmd
```

## Input Files

Input files are placed in `input/` before rendering the QMD. See
`docs/pipeline_overview.md` for how each file is generated.

**From the two-population Stacks run (6,074 SNPs):**

| File | Used in |
|------|---------|
| `populations.snps.vcf` | Kinship (Section 5) |
| `populations.fst_1-2.txt` | FST analysis (Section 1) |
| `populations.structure` | PCA, STRUCTURE (Sections 0.7, 0.8) |
| `Slep_LGs_MB.txt` | Coordinate conversion (Section 1) |
| `permuted_fst_values.txt` | Permutation test (Section 3) |
| `fst_bootstrap_values_combined.txt` | Bootstrap FST (Section 4) |
| `250630_genetic_pca.txt` | PCA plot (Section 2) |

**From the combined single-population Stacks run (6,310 SNPs):**

| File | Used in |
|------|---------|
| `populations_combined.snps.vcf` | LD pruning, Ne (Sections 0.3, 0.4) |
| `populations_pruned.bed/.bim/.fam` | Ne estimation (Sections 10, 11) |
| `Slep_full_LD.vcor` | LD analysis (Sections 7, 8, 9) |

**Created manually:**

| File | Used in |
|------|---------|
| `Slep_pop.txt`, `pop1.txt`, `pop2.txt` | Population maps (Section 0.2) |

> **Note:** `Slep_full_LD.vcor` (~1.5 GB) is not tracked in this repository.
> It can be regenerated from `populations_combined.snps.vcf` following
> Section 0.4 of the QMD.

## Raw Data

Demultiplexed RAD-seq reads for all individuals are deposited in the NCBI
Sequence Read Archive (SRA) under BioProject accession **PRJNA851781**.

The *Syngnathus leptorhynchus* reference genome is available in NCBI
GenBank under accession **GCA_054491065.1**.
