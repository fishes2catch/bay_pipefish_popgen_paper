# STRUCTURE Analysis Parameters

## Overview

Population genetic structure was investigated using STRUCTURE v2.3.4
(Pritchard et al. 2000) automated with StrAuto v0.3.1 (Chhatre & Emerson 2017).
Results were visualized using CLUMPAK (Kopelman et al. 2015).

> **Note on computing environment:** STRUCTURE runs were performed on the
> University of Oregon Genome computing cluster, which has since been
> decommissioned. Scripts are archived in `scripts/` for transparency and
> reproducibility documentation. StrAuto v0.3.1 requires **Python 2** and
> will not run on Python 3 without modification.

---

## STRUCTURE Run Parameters

| Parameter | Value |
|-----------|-------|
| K values tested | K = 1 to K = 4 |
| Runs per K | 5 |
| Burn-in | 100,000 steps |
| MCMC reps after burn-in | 40,000 steps |
| Model | Admixture, correlated allele frequencies |
| Parallelization | GNU Parallel (65% of available cores) |

---

## CLUMPAK Settings

Results from 5 replicate runs per K were processed using CLUMPAK
(http://clumpak.tau.ac.il/) to identify the major clustering mode and
generate the bar plot shown in Figure 3.

| Parameter | Value |
|-----------|-------|
| K reported | K = 2 (primary), K = 1 (comparison) |
| Alignment method | CLUMPP (Greedy algorithm) |
| Visualization | Bar plot, individuals ordered by population |
